### *About this project:*
*It is a small project of five console games. Implemented in _PHP 8.1.2_*

### *Requirements:*
* ___PHP 8.1.2 or never___
* ___Composer 2.5.4 or never___

### *Setup:*
* _Clone repository_
> git clone git@github.com:alex-p14/php-project-45.git
* _Go to directory_
> cd php-project-45
* _Install package_
> composer install
* _Run game_
> * _make brain-even_
> * _make brain-calc_
> * _make brain-gcd_
> * _make brain-progression_
> * _make brain-prime_
